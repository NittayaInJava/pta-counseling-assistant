# Online Pharmacy Counseling Assistant (WIP)

Demo project: Spring Boot backend for digital medication counseling based on e-prescriptions and patient history.

## Tech Stack
- Java, Spring Boot
- PostgreSQL (Docker)
- JPA / Hibernate
- REST API
- HTML / CSS / JavaScript (prototype frontend)

## Run locally
1. Start PostgreSQL via Docker
2. Start Spring Boot
3. Open: http://localhost:8080/
